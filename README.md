# LicitaNowChallenge

Este repositório contém a minha solução para o desafio técnico de refatorização da landing page da plataforma LicitaNow, implementada com HTML e CSS puros.

## Sobre o projeto

O objetivo foi refatorizar o design da landing page e implementar a interface com foco em:

- Estrutura HTML semântica
- Organização e escalabilidade do CSS
- Layout responsivo (desktop, tablet e mobile)

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript ( usado para implementar comportamento dinâmico da barra de navegação (mobile/desktop toggle))
